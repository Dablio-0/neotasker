## Clonando e Rodando o Projeto
Primeiramente, clone o repositório e navegue até seu diretório raíz:
```bash
git clone https://github.com/kenielf/neotasker.git
cd neotasker
```

Então, builde o projeto e o execute com o `mvn`:
```bash
mvn package
mvn -q exec:java
```
